# 业务词汇对照表

> LLM 自动维护。记录各 repo 中的业务术语，统一命名，标记不一致。

## 使用方式

- `/lcw-ingest` 摄入时自动提取业务词汇
- `/lcw-ingest-all` 完成后跨 repo 对照，标记同义词和冲突
- `/lcw-lint` 检查新代码是否引入了不一致的术语

### 状态说明

- **统一** — 各 repo 用法一致
- **不一致** — 存在别名或冲突，需关注
- **待定** — 含义尚不明确，需确认
